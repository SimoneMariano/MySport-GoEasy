  1. Capitolo 1 - Gestione dei Progetti

Creare un nuovo progetto

Per creare un nuovo progetto occorre aver ricevuto un token di licenza full.

Nella finestra iniziale di accesso, si preme il tasto in alto ("Nuovo Progetto")

Il nuovo progetto verrà in tal caso generato con i dati personali del titolare di licenza, archiviati localmente.

Nella finestra iniziale, viene chiesta la scelta dello sport. Tale scelta non è opzionale, in quanto molte voci di menu e la struttura delle tabelle dipendono dalllo sport scelto, così come i criteri per l'assegnazione del punteggio, e le statistiche.

La prima scehrmata che si apre all'utente è quella delle informazioni generali, che va compilata nel campi di interesse.

Una volta inseriti tutti i dati per la generazione di un progetto, bisogna premere il pulsante di salvataggio dati, ed il programma assegna al progetto un suo codice identificativo univoco, crea una cartella locale dedicata al progetto (scelta opzionale), e genera i file per la gestione del progetto stesso.

Tutti i dati saranno salvati in un file di estensione .myc, che è un semplice file di testo in formato csv. Senza questo file il progetto non viene caricato.

Il file .myc viene nominato col codice evento e con la denominazione evento. E' preferibile per quest'ultimo campo non usare stringhe di testo di lunghezza eccessiva, per semplicità d'uso. E' questo il file che consente l'apertura del progetto in modo corretto. Non va quindi assolutamente editato con i tool di windows, ma solo tramite il programma MySport Planner.
