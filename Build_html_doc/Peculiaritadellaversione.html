Peculiarità della versione

Versione 1.0

  1. Questa versone è progettata per funzionare in ambiente Windows, ed è stata verificata per W7 e W10. Si assume pertanto che ogni utente del programma abbia correttamente installato lo stesso nel proprio PC.
  2. Il programma è rilasciato mediante un file eseguibile di installazione. I dati di Progetto possono altresì essere salvati in una qualsiasi cartella del PC, oltre a quella del file eseguibile, scelta che viene salvata nei parametri di configurazione.
  3. Questa versione 1.0 del programma utilizza per ora solo Microsoft Outlook come client di posta. La gestione di altri client di posta è prevista nelle versioni successive, così come l'inclusione di metodi di condivisione delle informazioni più adoperati sulla rete (Dropbox, Facebook, Twitter, ecc ...).
  4. La validità del programma, nella versione di prova, è limitata ad un mese. sufficiente a valutare in modo esaustivo tutte le funzionalità. Possono essere concesse licenze con periodo di prova di durata maggiore e funzionalità quasi complete, per i beta tester.
